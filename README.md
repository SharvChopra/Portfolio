## üîß Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/SharvChopra/Portfolio.git
   cd Portfolio
   ```

2. **Setup Backend**
   ```bash
   cd backend
   npm install
   # Create .env file with MONGODB_URI and PORT
   node server.js
   ```

3. **Setup Frontend**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
